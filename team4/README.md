## 📋 Project Overview
 
The Gold Price Tracker allows users to monitor real-time gold prices through interactive charts, manage personal asset portfolios, and stay updated with the latest financial news — all with a clean UI and full dark mode support.
 
### ✨ Features
 
- 📈 Live gold price chart (1-month history via API)
- 💼 Asset management with add/remove functionality
- 📰 Integrated financial news feed
- 🔐 User authentication (Login & Register)

  ## 🛠️ Tech Stack
 
- **HTML5 / CSS3 / Vanilla JavaScript**
- **Chart.js** — gold price visualisation
- **News API** — financial news feed
- **LocalStorage** — client-side data persistence
 
- 🌙 Dark mode toggle
- 📱 Responsive design across all pages
